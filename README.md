Steps:
1. Rename this project "AI-Based-Social-Media-Management-and-Prediction-System" into "ajay"
2. Create new file "memory.json" in "/ajay/python3/python/" folder
3. Create new file ".env" in "/ajay/" folder AND Type GROQ_API_KEY=your_api_key (don't use quotes ', ", etc. )
4. You can use free api key of grok (Visit on free grok api)
5. Install Xampp
6. Double Tap on xampp-control.exe (which is in your xampp folder)
7. Run Apache and MySQL
8. Open any browser and Type: http://localhost:8080/ajay/
9. Open "http://localhost:8080/phpmyadmin/"
10. Create New Database "ajayembr" AND import "ajayembr_ajay_db.sql" file
